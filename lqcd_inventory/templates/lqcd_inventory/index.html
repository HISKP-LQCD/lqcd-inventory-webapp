LQCD Inventory

Purpose

Here you can find and index the computed perambulators. This website is a frontend for an underlying database. Add new data, edit existing data and delete data which is not available any more.

Usage

Click on the menu items above. You need to create dependent entries first; so create a storage and an ensemble first, then you can create an eigensytem entry.

Implementation

This website is built using Django, a Python library/framework for creating web applications. It comes with an object relational mapper (ORM) that automagically stores all the Python objects in a relational database.

Design of the website is done with the Bootstrap CSS Framework. No CSS had to be written by hand, it is also responsive and mobile friendly for free.
